# 社区物业管理系统业务流程文档

## 系统角色

1. **系统管理员（Admin）**
2. **物业管理员（Manager）**
3. **业主（Owner）**
4. **维修人员（Worker）**

---

## 业务模块概览

### 核心业务模块（8 个）

1. ✅ **用户管理** - 账号、权限、角色
2. ✅ **住户与房产管理** - 住户信息、房产信息、入住搬离
3. ✅ **费用管理** - 账单生成、在线缴费、欠费催缴
4. ✅ **虚拟钱包管理** - 钱包充值、余额查询、支付密码
5. ✅ **投诉管理** - 投诉提交、处理、反馈、评价
6. ✅ **维修管理** - 报修、派工、维修、验收
7. ✅ **公告管理** - 发布、查看、统计
8. ✅ **停车管理** - 车位管理、车位租赁

### 辅助功能模块（2 个）

9. ✅ **统计报表** - 费用统计、服务质量分析
10. ✅ **系统维护** - 日志、权限管理

---

## 一、用户管理流程

### 1.1 用户注册与登录流程

**【系统管理员】**

创建用户账号 → 分配角色 → 设置初始密码

**【所有用户】**

登录系统 → 身份验证（JWT） → 获取权限 → 访问对应功能

---

## 二、住户与房产管理流程

### 2.1 房产信息录入流程

**【物业管理员】**

创建楼栋信息 → 创建单元信息 → 创建房间信息 → 设置房产属性  
（建筑面积、户型、状态等）

### 2.2 住户入住流程

**【物业管理员】**

登记住户基本信息 → 绑定房产 → 创建业主账号 →  
设置住户类型（业主/租户） → 初始化虚拟钱包（余额为 0） →  
通知业主账号信息

**【业主】**

首次登录 → 修改登录密码 → 设置支付密码（6 位数字） →  
完善个人信息

### 2.3 住户搬离流程

**【业主】**

提交搬离申请

**【物业管理员】**

审核申请 → 检查欠费情况 → 检查钱包余额 →  
结清所有费用 → 处理钱包余额退款 →  
更改住户状态为“已搬离” → 解绑房产关系 →  
冻结业主账号

---

## 三、虚拟钱包管理流程 💰

### 3.1 支付密码设置流程

**【业主 - 首次登录】**

进入个人中心 → 点击“设置支付密码” →  
输入 6 位数字支付密码 → 确认支付密码 →  
提交设置 → 设置成功

**业务规则：**

- 支付密码必须为 6 位数字
- 不能与登录密码相同
- 首次登录必须设置支付密码才能使用钱包功能

### 3.2 支付密码修改流程

**【业主】**

进入个人中心 → 点击“修改支付密码” →  
输入原支付密码 → 输入新支付密码（6 位数字） →  
确认新支付密码 → 提交修改 → 修改成功

**业务规则：**

- 必须验证原支付密码
- 新密码不能与原密码相同
- 记录修改日志

### 3.3 支付密码重置流程

**【业主】**

点击“忘记支付密码” → 选择验证方式 →  
身份验证（手机验证码-输入手机号发送验证码，验证码在服务器后台直接打印，模仿验证码流程） →  
设置新支付密码 → 确认新密码 → 重置成功

**【系统管理员】**

接收业主申请 → 验证业主身份 →  
重置支付密码为初始密码 → 通知业主

### 3.4 虚拟钱包充值流程（模拟充值）

**【业主】**

进入“我的钱包” → 查看当前余额 →  
点击“充值” → 输入充值金额 →  
输入支付密码 → 验证支付密码 →  
充值成功 → 更新钱包余额 →  
生成充值记录（记录时间、金额、余额）

**业务规则：**

- 单次充值金额：0.01 元 - 10000 元
- 充值金额精确到分（小数点后 2 位）
- 充值即时到账
- 记录充值流水号
- 钱包余额上限：100000 元

### 3.5 钱包余额查询流程

**【业主】**

进入“我的钱包” →  
查看当前余额 →  
查看充值记录 →  
查看消费记录（支付账单记录）

**显示信息：**

- 当前余额
- 累计充值金额
- 累计消费金额
- 最近 10 条交易记录

### 3.6 钱包交易明细查询流程

**【业主】**

进入“交易明细” →  
选择查询时间范围 →  
选择交易类型（充值/消费/全部） →  
查看交易列表 →  
点击查看交易详情

**交易详情包含：**

- 交易流水号
- 交易时间
- 交易类型（充值/消费）
- 交易金额
- 账单信息（如果是消费，才有账单信息）
- 交易后余额

---

## 四、费用管理流程 💳

### 4.1 费用标准设置流程

**【物业管理员】**

创建费用类型（物业费/水费/电费/停车费等） →  
设置收费标准（单价、计费周期） →  
设置生效日期 → 保存费用标准

### 4.2 账单生成流程

**【物业管理员】**

选择账单周期（如 2025 年 1 月） →  
选择费用类型（物业费、水电费等） →  
系统自动计算各住户应缴金额 →  
批量生成账单 →  
账单状态设置为“待缴费” →  
（可选）生成欠费通知

**【系统自动】**

定时任务每月 1 号自动生成上月账单

**账单包含信息：**

- 账单编号（唯一）
- 业主信息
- 费用类型
- 费用周期
- 应缴金额
- 账单状态（待缴费/已缴费/已超期）
- 生成时间
- 缴费截止日期

### 4.3 在线缴费流程（虚拟钱包支付）⭐

**【业主】**

登录系统 → 进入“我的账单” →  
查看待缴费账单列表 →  
选择要缴纳的账单（可多选） →  
查看总金额 → 点击“支付” →  
系统检查钱包余额是否充足 →  
如果余额不足：提示充值 →  
如果余额充足：输入支付密码 →  
验证支付密码 →  
扣除钱包金额 →  
更新账单状态为“已缴费” →  
生成缴费记录 →  
生成电子收据 →  
显示支付成功页面

**业务规则：**

- 支持单笔或批量缴费
- 批量缴费时按账单生成时间排序依次扣款
- 如果批量支付中途余额不足，已支付的不退回，未支付的保持原状态
- 支付成功后实时更新账单状态
- 支付失败（密码错误/余额不足）提示明确的错误信息

**状态变更：**

待缴费 → 已缴费

### 4.4 电子收据生成流程 📄

**【系统自动】**

支付成功后自动生成电子收据

**电子收据包含：**

- 收据编号（唯一）
- 收据生成时间
- 业主信息（姓名、房号、联系方式）
- 账单信息（账单编号、费用类型、费用周期）
- 应缴金额
- 实缴金额
- 缴费时间
- 物业公司盖章（电子章）

**【业主】**

查看我的账单 → 点击“已缴费”标签 →  
选择账单 → 点击“查看收据” →  
在线查看电子收据 →  
可下载 PDF 格式收据 →  
可打印收据

### 4.5 费用查询流程

**【业主】**

登录系统 → 查看“我的账单” →  
查看待缴费账单（状态：待缴费） →  
查看已缴费账单（状态：已缴费） →  
查看账单详情 →  
下载电子收据

**筛选条件：**

- 按时间范围
- 按费用类型
- 按缴费状态

**【物业管理员】**

查询所有账单 →  
按楼栋/住户/时间/缴费状态筛选 →  
查看账单详情 →  
查看缴费记录 →  
导出 Excel 报表

### 4.6 欠费催缴流程

**【系统自动】**

每日检查超期未缴费账单 →  
更新账单状态为“已超期” →  
生成催缴通知

**【物业管理员】**

查看欠费统计 →  
筛选逾期欠费住户 →  
生成催缴通知 →  
系统记录催缴日期和次数 →  
发送催缴通知（站内信）

**【业主】**

登录系统看到欠费提醒（红色标记） →  
查看欠费明细 →  
可立即在线缴费

**状态说明：**

- 待缴费：未到缴费截止日期
- 已超期：超过缴费截止日期但未缴费

## 五、投诉管理流程

### 5.1 投诉提交流程

**【业主】**

登录系统 → 填写投诉单 →  
选择投诉类型（环境卫生/噪音/设施损坏/服务态度等） →  
填写投诉内容 →  
上传图片（可选，最多 5 张） →  
选择紧急程度（普通/紧急） →  
提交投诉 →  
投诉状态：待处理

### 5.2 投诉处理流程

**【物业管理员】**

查看待处理投诉 →  
查看投诉详情 →  
评估投诉类型和紧急程度 →  
分配处理人员（选择物业管理员） →  
记录处理措施 →  
更新处理状态为“处理中” →  
跟进处理进度 →  
填写处理结果 →  
上传处理照片（可选） →  
标记为“已处理” →  
通知业主

**处理状态流转：**

待处理 → 处理中 → 已处理 → 已关闭

### 5.3 投诉反馈流程

**【业主】**

收到处理完成通知 →  
查看处理结果 →  
查看处理照片 →  
进行满意度评价 →  
选择评价等级（满意/一般/不满意） →  
填写评价意见（可选） →  
提交评价 →  
投诉状态更新为“已关闭”

**业务规则：**

- 业主评价后投诉自动关闭
- 如 7 天内未评价，系统自动关闭并默认为“满意”

---

## 六、维修管理流程

### 6.1 报修申请流程

**【业主】**

登录系统 → 提交报修申请 →  
选择维修类型（水电/门窗/电梯/公共设施等） →  
描述故障情况 →  
上传故障图片（可选，最多 5 张） →  
选择紧急程度（普通/紧急） →  
预约维修时间（可选） →  
提交申请 →  
维修单状态：待派工

### 6.2 维修派工流程

**【物业管理员】**

查看待派工的维修单 →  
查看维修详情和故障描述 →  
评估维修难度和所需时间 →  
选择合适的维修人员 →  
分配维修任务 →  
设置要求完成时间 →  
维修单状态更新为“已派工” →  
通知维修人员

**【维修人员】**

登录系统 → 查看“我的维修任务” →  
查看维修单详情 →  
查看故障描述和图片 →  
查看业主联系方式 →  
确认接单 →  
联系业主预约时间

### 6.3 维修执行流程

**【维修人员】**

更新维修状态为“进行中” →  
到现场维修 →  
拍摄故障现场照片 →  
记录实际故障原因 →  
记录维修措施 →  
更换配件（如有，记录配件名称和数量） →  
拍摄维修后照片 →  
更新维修状态为“待验收” →  
通知业主验收

**维修状态流转：**

待派工 → 已派工 → 进行中 → 待验收 → 已完成

### 6.4 维修验收流程

**【业主】**

收到验收通知 →  
查看维修进度 →  
查看维修前后照片对比 →  
查看维修内容说明 →  
进行验收评价 →  
选择验收结果（合格/不合格） →  
评分（1-5 星） →  
填写评价意见 →  
确认完成

**【物业管理员】**

查看验收结果 →  
如不合格：

- 查看不合格原因
- 重新派工或指派同一维修人员整改
- 维修单状态回退到“进行中”

如合格：

- 更新维修单状态为“已完成”
- 记录维修费用（如有）
- 归档维修记录
- 维修人员工作量统计+1

**业务规则：**

- 业主 3 天内未验收，系统自动验收为“合格”
- 不合格维修单最多返工 2 次
- 维修完成后记录到维修人员工作记录

## 七、公告管理流程

### 7.1 公告发布流程

**【物业管理员】**

进入公告管理 → 点击“新建公告” →  
选择公告类型（通知/停水停电/活动/政策等） →  
输入公告标题 →  
编辑正文内容（支持富文本编辑器） →  
上传附件（可选，支持 PDF/Word/图片） →  
设置是否置顶 →  
选择发布范围：

- 全部住户
- 指定楼栋
- 指定单元

设置有效期（开始时间-结束时间） →  
预览公告 →  
发布公告 →  
公告状态：已发布

**业务规则：**

- 置顶公告最多 5 条
- 公告标题最多 50 字
- 正文最多 5000 字
- 附件单个最大 10MB，总共最多 3 个

### 7.2 公告查看流程

**【业主】**

登录系统 → 进入“社区公告” →  
查看公告列表（按发布时间倒序，置顶在前） →  
未读公告显示红点标记 →  
点击查看公告详情 →  
系统自动标记为已读 →  
系统记录阅读时间 →  
可下载附件

**筛选功能：**

- 按公告类型筛选
- 搜索公告标题

### 7.3 公告管理流程

**【物业管理员】**

查看公告列表 →  
查看公告详情 →  
查看公告阅读统计：

- 应读人数
- 已读人数
- 阅读率
- 未读人员名单

编辑公告（只能编辑未过期的公告） →  
取消置顶 →  
删除公告（软删除，改为“已撤回”状态） →  
导出阅读统计报表

**公告状态：**

- 已发布：正常展示
- 已过期：超过有效期，不在列表显示但可查询
- 已撤回：管理员删除，不展示

---

## 八、停车管理流程 🅿️

### 8.1 车位信息管理流程

**【物业管理员】**

录入车位信息 →  
设置车位编号 →  
设置车位状态（空闲/已租） →  
设置停车费标准

### 8.2 车位租赁流程

**【业主】**

提交车位租赁申请 →  
选择车位 →  
填写车辆信息（车牌号、车型等）

**【物业管理员】**

审核申请 →  
确认车位可用 →  
批准申请 →  
生成租赁合同 →  
更新车位状态 →  
关联业主与车位 →  
生成停车费账单（按月/季/年）

### 8.3 车位续租流程

**【业主】**

租期到期前收到续租提醒 →  
查看当前租赁信息 →  
选择续租期限 →  
查看续租金额 →  
提交续租申请 →  
生成续租账单 →  
使用钱包支付 →  
支付成功 →  
租期自动延长

**【系统自动】**

租期到期前 7 天：发送续租提醒

租期到期：

- 如已续租：租期延长
- 如未续租：车位状态改为“空闲”，解除绑定关系

### 8.5 车位退租流程

**【业主】**

进入“我的车位” →  
选择要退租的车位 →  
提交退租申请 →  
说明退租原因（可选）

**【物业管理员】**

查看退租申请 →  
检查停车费缴纳情况 →  
如有欠费：通知业主结清 →  
如无欠费：审核通过 →  
计算剩余租期 →  
如需退款：

- 计算退款金额（按剩余天数比例）
- 退款到业主钱包
- 生成退款记录

解除车位绑定 →  
更新车位状态为“空闲” →  
通知业主退租成功

**业务规则：**

- 提前退租按剩余天数的 70%退款
- 退款金额返回到业主虚拟钱包
- 租期不足 1 个月不予退款

## 九、统计报表流程

### 9.1 费用统计流程

**【物业管理员】**

进入“统计分析” → 选择“费用统计” →  
选择统计周期（日/月/季/年） →  
选择统计维度：

- 按楼栋统计
- 按费用类型统计
- 按缴费状态统计

生成统计图表（可切换）：

- 柱状图
- 饼图
- 折线图

查看统计数据 →  
导出 Excel 报表 →  
打印报表

**统计内容：**

- 费用收缴率
- 欠费金额统计
- 收入趋势分析
- 各楼栋缴费对比

## 十、系统维护流程

### 10.1 用户管理流程

**【系统管理员】**

进入“用户管理” →  
创建用户：

- 输入用户名、真实姓名、手机号
- 选择角色（管理员/物业管理员/业主/维修人员）
- 设置初始密码
- 如果是业主：关联房产信息
- 保存用户

编辑用户：

- 修改用户信息
- 变更角色
- 启用/禁用账号

删除用户：

- 软删除（账号状态改为“已删除”）
- 数据保留但无法登录
